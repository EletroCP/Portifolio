Desenvolvimento

Meu método de desenvolvimento é dividido em 4 etapas a de estruturação, testes, implementação e refatoração.

Primeiro, crio a estrutura base do projeto para ajudar o na etapa de testes e implementação.

para isso uso ferramentos como o Figma que posibilitam uma prévia do modelo do site.

Esse é o exemplo do figma para desktop.

Em seguida passo para os testes, eles servem para verificar se os elementos estão todos funcionando como deveriam.
