Desenvolvimento

Meu método de desenvolvimento é dividido em 4 etapas a de estruturação, testes, implementação e refatoração.

Primeiro vou contruir a estrutura base do projeto para ajudar na etapa de testes e implementação.

Para isso uso ferramentas como o Figma, que posibilitam uma prévia do modelo do site.

Esse é o exemplo do figma para desktop.

Em seguida os testes, eles servem para verificar se os elementos estão todos funcionando.

Hora de por a mão na massa.

Primeiro vem o HTML depois o JavaScript e por ultimo o CSS.

Considerados esqueleto, cérebro, e corpo do site.

Então refatorar, rodar os testes de novo para certificar o funcionamento, verificar se não há nada fora do lugar e finalizar mais um tabalho.
