Desenvolvimento

Meu método de desenvolvimento é dividido em 4 etapas a de estruturação, testes, implementação e refatoração.
Primeiro, crio a estrutura base do projeto para ajudar o na etapa de testes e implementação, para isso uso ferramentos como o Figma que posibilitam uma prévia do modelo do site.

Em seguida passo para os testes, eles servem para verificar se os elementos estão todos funcionando como deveriam.

Hora de por a mão na massa, começo implementando o HTML, ele é responsavel pela estrutura basica do site como textos, imagens, links e "containeres" que ajudam na hora de estilizar o site, Essa parte é feita com CSS, Cores e efeitos, e tambem na responsividade para que a pagina funcione nos mais diversos dispositivos. E então o Javascript, a parte lógica, em que permite a manipulação dos elementos de forma facil e dinâmica.

E por ultimo e provavelmente mais importante a refatoração, após adicionar os elementos mover eles para cima e para baixo e adicionar lógica, existe a possibilidade de uma imagem estar sobrepondo um botão, ou ter faltado de por um link em algum lugar. É o momento de rodar os testes novamente para certificar o funcionamento, verificar se não há nada fora do lugar para então finalizar mais um tabalho.
